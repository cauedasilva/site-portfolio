# Site Portfolio

## Visão Geral

Este repositório contém o código do meu portfólio pessoal.

O site é uma SPA com foco em responsividade.

## Tecnologias e Bibliotecas

O site foi construído utilizando uma stack front-end em vite.

### Tecnologias Principais
* **JavaScript:** Linguagem principal.
* **React:** Utilizado para construir a interface.
* **Vite:** Usado como ferramenta de build e servidor de desenvolvimento.
* **HTML5**
* **CSS3**

### Estas são as bibliotecas utilizadas para estilização e funcionalidade:

* **Tailwind CSS:** Framework de CSS.
* **EmailJS:** Serviço utilizado para enviar e-mails diretamente do formulário de contato do cliente, sem back-end.
* **React-Toastify:** Biblioteca para exibir notificações na aplicação (toasts).
